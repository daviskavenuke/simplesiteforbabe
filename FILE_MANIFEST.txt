DATABASE SCHEMA
================================================================================

Product Model:
- id: MongoDB ObjectId
- name: String (unique)
- description: String (10-2000 chars)
- price: Float (> 0)
- category: String
- image: String (optional)
- createdAt: DateTime (auto)
- updatedAt: DateTime (auto)

Admin Model:
- id: MongoDB ObjectId
- email: String (unique)
- password: String
- name: String (optional)
- createdAt: DateTime (auto)
- updatedAt: DateTime (auto)

ENVIRONMENT VARIABLES
================================================================================

Required:
- NEXTAUTH_URL .................. Auth URL (http://localhost:3000)
- NEXTAUTH_SECRET ............... Secret key (generate with openssl)
- DATABASE_URL .................. MongoDB connection string

Admin Setup:
- ADMIN_EMAIL ................... Admin email
- ADMIN_PASSWORD ................ Admin password

WhatsApp Integration:
- NEXT_PUBLIC_WHATSAPP_NUMBER ... WhatsApp phone number

Optional:
- NEXT_PUBLIC_CLOUDINARY_CLOUD_NAME
- CLOUDINARY_API_KEY
- CLOUDINARY_API_SECRET

QUICK REFERENCE

Start Development:
  npm install
  npm run dev

Setup Database:
  npx prisma generate
  npx prisma migrate dev --name init
  npm run prisma:seed

View Database:
  npx prisma studio

Type Checking:
  npx tsc --noEmit

Build for Production:
  npm run build
  npm start
